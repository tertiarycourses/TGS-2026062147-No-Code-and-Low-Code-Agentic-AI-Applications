Payload JSON Schema

The payload is dispatched directly using the following format:

{
  "event": "document_dispatch",
  "timestamp": "2026-06-24T05:20:00Z",
  "documents": [
    {
      "name": "manual.pdf",
      "size": 124500,
      "type": "application/pdf",
      "text": "Parsed plain text extracted from PDF content pages...",
      "base64": "data:application/pdf;base64,JVBERi0xLjQK..."
    }
  ]
}

Ideal for workflow processors like n8n, Pipedream, Zapier or server endpoints that decode base64 back into documents.

Developer Advice

If you run into CORS policy blockages in your browser, ensure that your webhook service headers allow access from all origins, or utilize tools like n8n's raw webhook Node options.
